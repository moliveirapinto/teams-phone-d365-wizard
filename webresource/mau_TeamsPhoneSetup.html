Before you start — phone number prerequisites
This guide does not provision Teams phone numbers. You must already have a number assigned in Teams, and it must be of type Call Queue or Auto Attendant. User (DID) numbers are not supported by the Voice channel onboarding flow.

Troubleshooting
"Instance not found" on Set-CsOnlineApplicationInstance — wait 60–120s after creating the resource account for AAD propagation. The script already includes a 60s wait.

Sync from Teams returns 500 / nothing — re-run the script. It re-checks (and creates if missing) the Entra app registration with TeamsResourceAccount.Read.All + admin consent that "Sync from Teams" requires.

Calling Plan numbers — only service numbers are supported. Acquire one in Teams admin center first, then assign it to this resource account through the Teams admin UI (the script skips Set-CsPhoneNumberAssignment for this path).
